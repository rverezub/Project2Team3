Illustrations of Improvements

Below are visual representations showcasing how the identified issues were addressed through design improvements. Each point from the improvement discussion page is paired with a "Before" and "After" image to highlight the changes made.

1. Color Coding

Before: Inconsistent color coding that made it difficult to link related courses.

After: Linked courses now share the same color, improving user clarity.

2. Lab/Recitation Prompt

Before: No automatic prompts to link labs or recitations, leading to missed steps.

After: Automatic prompts guide users to link labs or recitations, ensuring smoother registration.

3. Error Messages

Before: Overly technical error messages displaying raw HTML, confusing users.

After: Simplified error messages with clear, actionable feedback.

4. Schedule Auto-Load

Before: Schedule defaulted to 12 a.m., requiring manual adjustment to view registered classes.

After: Schedule automatically loads to the first registered class, saving time and effort.

5. Course Search

Before: Separate inputs for course name and number added complexity.

After: A unified search bar simplifies and speeds up the search process.

6. Navigation and Tabs

Before: Poorly structured tabs with repetitive functions overwhelmed users.

After: Tabs are now streamlined and clearly labeled, improving navigation efficiency.

7. Remove Class Action

Before: Removing classes required selecting an option from a dropdown menu.

After: Classes can now be removed with a single, clearly visible button.

8. Schedule Table

Before: Redundant information cluttered the schedule table, overwhelming users.

After: The table has been simplified to include only essential details.

9. Registration PIN Requirement

Before: Users were required to repeatedly enter their registration PIN, causing frustration.

After: Users can now stay logged in, eliminating the need for repeated PIN entries.

10. Help Resources

Before: Outdated or hard-to-access help resources left users without guidance.

After: Updated and easily accessible help resources now support users effectively.

These improvements collectively enhance the usability and efficiency of the Illinois Tech registration portal, addressing user frustrations and creating a more seamless experience.
